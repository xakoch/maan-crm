# Дашборд дилера

## Доступ

Роль: `dealer`
Базовый путь: `/dashboard/dealer`

---

## 1. Dashboard `/dashboard/dealer`

### Статистика

Карточки (только свои данные):

- Всего лидов
- Закрыто сделок
- Конверсия (%)
- Выручка

### График

- Лиды по дням (последние 30 дней)

### Топ-3 менеджера

Таблица:

- ФИО
- Кол-во лидов
- Закрыто
- Конверсия

---

## 2. Лиды `/dashboard/dealer/leads`

### Таблица (только свои лиды)

Колонки:

- ID
- Имя
- Телефон
- Менеджер
- Статус
- Источник
- Дата
- Действия

### Фильтры

- Статус
- Менеджер (только свои)
- Дата

### Поиск

- По имени/телефону

### Действия

- Просмотр: `/dashboard/dealer/leads/[id]`
- Назначить менеджера
- Изменить статус

---

## 3. Детали лида `/dashboard/dealer/leads/[id]`

Аналогично супер-админу, но:

- Нельзя удалить лид
- Нельзя переназначить на менеджера другого дилера

---

## 4. Менеджеры `/dashboard/dealer/managers`

### Таблица (только свои менеджеры)

Колонки:

- ФИО
- Telegram
- Статус привязки
- Кол-во лидов
- Активен
- Действия

### Кнопка "Добавить менеджера"

→ модалка с формой

### Действия

- Редактировать: `/dashboard/dealer/managers/[id]`
- Деактивировать

---

## 5. Профиль `/dashboard/dealer/profile`

### Информация о филиале

- Название
- Город / Регион
- Адрес
- Владелец (имя, телефон, email)
- Статус

### Кнопка "Редактировать"

Модалка с формой (только контакты, нельзя менять город/название)

---

## Ограничения

- Дилер видит ТОЛЬКО свои лиды/менеджеров
- Не может видеть других дилеров
- Не может удалять лиды
- RLS в Supabase автоматически фильтрует по `tenant_id`
